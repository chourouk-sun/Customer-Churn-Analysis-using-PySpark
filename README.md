# Customer-Churn-Analysis-using-PySpark
### Project Goal
Use PySpark to:
Process a large dataset
Clean & transform data
Perform analytics with Spark SQL
Build a machine learning model (MLlib)
Evaluate churn prediction
Tools (IMPORTANT)
✅ PySpark
✅ Jupyter Notebook
✅ Spark SQL
❌ NO Eclipse
❌ NO Scala

 Dataset
 https://www.kaggle.com/datasets/blastchar/telco-customer-churn?utm_source=chatgpt.com 
Use Telco Customer Churn (very popular & realistic)
Columns:
customerID
tenure
MonthlyCharges
TotalCharges
Contract
PaymentMethod
Churn
